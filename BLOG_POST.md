## 🏗️ I Built a Free, Open-Source AI Architect Learning Path — 10 Hands-On Notebooks Covering Everything from LLMs to Production Agents

After months of studying AI architecture, I realized there's no single resource that teaches you **everything** — from transformer theory to deploying RAG systems in production.

So I built one. **Completely free. No paid APIs needed.**

### 🔗 GitHub: https://github.com/rahgoel2510/ai-architect-portfolio

---

### What's Inside (10 Jupyter Notebooks):

📓 **01 — LLM Evaluation**: Transformer internals, attention mechanism, benchmarking models with statistical significance testing

📓 **02 — Model Selection**: Pareto frontiers, weighted scoring, TCO analysis, scaling laws

📓 **03 — Vector Databases**: HNSW internals, FAISS vs ChromaDB vs Pinecone, chunking strategies

📓 **04 — RAG Pipeline**: Naive → Advanced RAG, HyDE, multi-query retrieval, RAGAS evaluation

📓 **05 — MLOps**: MLflow tracking, KS-test drift detection, CI/CD pipelines, LLMOps

📓 **06 — Cloud Deployment**: AWS Lambda + API Gateway, SAM templates, cold start optimization

📓 **07 — Observability**: Prometheus metrics, OpenTelemetry tracing, SLOs & error budgets

📓 **08 — Compliance**: GDPR, EU AI Act, PII detection, privacy-preserving RAG

📓 **09 — Prompt Engineering**: Structured output, injection defense, bias testing, prompt testing frameworks

📓 **10 — Agentic Systems**: ReAct agents, LangGraph, multi-agent routing, guardrails

---

### Why This Is Different:

✅ **Theory + Code** — Each notebook teaches concepts THEN implements them
✅ **100% Free** — Uses OpenRouter free tier (LLaMA 3.1, Gemma 2) + local embeddings
✅ **Production-Ready Patterns** — Not toy examples, real architecture decisions
✅ **Run Anywhere** — Google Colab, GitHub Codespaces, or locally
